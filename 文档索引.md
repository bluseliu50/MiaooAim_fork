## 按角色查找

| 角色 | 推荐阅读顺序 |
|------|----------------|
| 复刻 / 首次烧录 | README（引脚 / Flash）→ `hardware/BOM*.md` 提要 |
| 二次开发 / AI 助手 | AGENTS.md → `main/` 源码 |
| 课程表用户 | COURSE_TABLE_README → README（Web 入口） |

## 按问题查找

| 问题 | 文档 |
|------|------|
| 定时唤醒后打不开网页 | README / AGENTS.md **双启动路径** |
| Basic Auth / 天气 Key 不透传 | AGENTS.md **HTTP subsystem**、配置页说明 |
| 图片/抖色/内存 OOM | AGENTS.md **Rendering pipeline**、`main/image_convert.c` |
| HTTP 路由清单 | `main/http_app.c` 内 `uris[]`（约 63 条） |
| 课程表 JSON | COURSE_TABLE_README、`feat_timetable_*` |
| 网页推屏 canceled | AGENTS.md **Display arbitration** |
